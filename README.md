# lmcli

A CLI tool for chatting with multiple LLM providers (Anthropic, OpenAI, etc.) from a single interface.

## Features
- **Multi-provider support**: Use Anthropic, OpenAI, Groq, and other compatible APIs from one tool
- **Account management**: Store multiple API keys and switch between them easily
- **Conversation history**: Automatic context tracking with configurable message limits
- **Simple configuration**: Human-readable JSON config files
- **Cross-platform**: Works on Linux and Windows (via WSL)
